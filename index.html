# 欢迎使用 Luban

# 介绍

Luban 是一个强大的生成与缓存工具。最初只用于对象生成，对象可以是常规代码、配置数据、类似protobuf的消息代码，也可以是游戏资源如assetbundle。 在大型项目中，由于配置或资源数据庞大，生成对象可能会花费相当多的时间。比如一个典型的MMORPG项目，后期全量生成配置，即使用了多线程加速，所需时间 也在10秒的级别。因此使用client/server模式配合缓存机制来大幅加速生成过程。

Luban 最初是为了解决传统excel导出工具功能过于薄弱，无法很好解决MMORPG游戏复杂配置需求的痛点问题。 自2015年以来，经历过 MMORPG、卡牌、SLG 等多个上线项目的考验， 实际项目过程中不断迭代和优化，理解逐渐深入，最终由一个增强型的配置工具成为一个 相对完备的游戏配置数据解决方案。
